Conociendo PWA

¿Que es PWA y que lenguajes abarca?

Progressive web apps (PWA) o aplicación web progresiva es una solución basada en la web tradicional que todos conocemos, aunque incorpora algunas particularidades que la hacen parecerse a una app nativa para teléfonos móviles y tabletas. Cada proyecto es independiente y el nivel de adaptación de la web al formato app es progresivo, por lo que puede que visitemos un site con una PWA casi idéntica a una app o quizá sea una web con algunos avances. El lenguaje base que utiliza PWA son: HTML, CSS y JavaScript.

¿Que fue el creador del concepto PWA?

La idea de un PWA no fue inventada ni por Google ni por Apple, pero fue Steve Jobs quien presentó el concepto por primera vez frente al mundo, durante la presentación del iPhone en 2007.

¿Cuáles son los elementos principales y Secundarios de PWA?

El Manifest es el elemento principal e imprescindible del framework PWA que se llama “Application Manifest” este archivo se compone de: Nombre de la PWA, Icono, Color principal, Idioma principal, Como deseamos visualizarla y una breve descripción de que trata la página. Y los elementos secundarios tenemos: service worker, diseño responsive, API de geolocalización y API media.
